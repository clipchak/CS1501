Project 2 Submission for CS1501


Files given and used: 
AnagramDecoder.java, LinkedHashEntry.java, StringTable.java, dictionary.txt


This program works in the command line as 

 	 java AnagramDecoder DICTIONARY ANAGRAM

For example, 

	 java AnagramDecoder dictionary.txt ltse

Produces the output 

	Possible word: lets

and works for any word provided in the dictionary.txt file by reading
the words in and placing them into a hash table. 

Program issues: 
Does not work with multiple word anagrams.
